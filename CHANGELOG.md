# Changelog

## 0.1.0

- Initial release.
- **Invofox API** credentials (API key via the `x-api-key` header) with a connection test.
- Three operations:
  - **Upload** — upload one or more binary files (multipart) with optional metadata.
  - **Upload by URL** — upload one or more files by their public URL.
  - **Get** — retrieve a document with its status and extracted data, with an optional simplified output.
